Cardgen
=======

To build an example, run makefile to generate design pattern cards from intents.txt


Cardgen Usage
-------------

For help:

	./cardgen.py -h 

To run:

	./cardgen.py -i inputfile -o outputfile

or using stdin/stdout:

	./cardgen.py < inputfile > outputfile
	

Inputfile Structure
-------------------

	First Card:
	Definition Line 1
	Definition Line 2
	
	Second Card:
	Single Line Definition
